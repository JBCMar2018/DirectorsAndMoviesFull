Directors & Movies

This is an example of the director/movie application that includes fragments. Have a look through the code. There shouldn't be anything at this point that you are not already familiar with. This example demonstrates how to use a one to many relationship (between Directors and Movies). Change it to a many to many relationship, and include actors. Have fun at the movies!

Styling

This project uses the bootstrap grid, and so it should be responsive. Make your browser winder smaller to see how the application should render on smaller screens. You can also change the phone type (Google Chrome), to see how the application will render on different preset screen sizes, or you can change the screen size if necessary.

For the best experience, when you're on your home network, browse to your computer's IP address :8080, and look at your application!
